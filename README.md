# Palliative Care Volunteer Coordination

**International comparison of palliative care volunteer coordination frameworks, guidelines, and operational models — with localized versions by country.**

---

## 🌍 Project Overview

This repository documents how different nations structure, govern, and operate volunteer coordination within hospice and palliative care settings. It is designed to support cross-border learning, policy development, and the creation of localized coordination materials.

## 🔑 Key Coordination Materials

Each country folder contains:
- **coordination-framework.md** — National framework, governance, and legal context
- **volunteer-model.md** — Operational model for volunteer recruitment, training, and deployment
- **guidelines.md** — Practice guidelines and quality standards

## 🌐 Translated Materials

Key coordination materials are translated into Spanish, French, and German under `translations/`. A Lara Translate translation memory (`palliative_care_volunteer_coordination`) is maintained to ensure consistency as the project grows.

## 📖 How to Contribute

1. Pick a country not yet covered
2. Use the template in `templates/coordination-handbook-template.md`
3. Submit a PR with your country's folder
4. If translating, use the translation memory for consistency
